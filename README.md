2026-bot (mbbots)
"Aut viam inveniam aut faciam."
Status
Current State: Pre-Alpha / Broken
Swerve base is initialized but untested.
YAGSL configuration is present.
Major subsystems are pending implementation.
 Prerequisites & Installation
Required Software
Java JDK 17
WPILib 2025
Vendor Dependencies:
Phoenix 6 (CTRE)
REVLib (REV Robotics)
YAGSL (Swerve)
Setup
Clone the repository and install dependencies via Gradle:
git clone [https://github.com/your-username/2026-bot.git](https://github.com/your-username/2026-bot.git)
cd 2026-bot
./gradlew build


 Build & Deploy
Build the code:
./gradlew build


Deploy to Robot:
Ensure you are connected to the robot radio (Wi-Fi or Ethernet).
./gradlew deploy
